# Mein Portfolio - Lennart Kamp Gonzakez

## Beschreibung
Dies ist meine erste einfache Webseite.
Sie enthält Informationen über mich, meine Skills
und ein Kontaktformular.

## Verwendete Techniken
- HTML5 (Struktur der Webseite)
- CSS (Styling der Seite)
- Flexbox für die Navigation
- Hover-Effekte für Links und Button

## Weg
Ich habe die Weg B variante (Portfolio-Seite) gewählt.

## Notizen
Ich habe gelernt, wie man eine Webseite mit HTML strukturiert
und mit CSS gestaltet. Besonders hilfreich war Flexbox
für die Navigation und das Styling des Formulars.
